# Quick Keybindings Summary

## Most Important (Daily Use)

| Action | Keybind | What It Does |
|--------|---------|--------------|
| **Toggle Sidebar** | `Ctrl+B` | Show/hide file explorer |
| **Find Files** | `Ctrl+P` | Fuzzy search & open files |
| **Next Tab** | `Ctrl+Tab` | Switch to next open file |
| **Prev Tab** | `Ctrl+Shift+Tab` | Switch to previous file |
| **Close Tab** | `Ctrl+W` | Close current file |
| **Open Terminal** | `Ctrl+~` (backtick) | Toggle terminal |
| **Go to Definition** | `gd` | Jump to function/variable definition |
| **Rename** | `F2` or `<leader>rn` | Rename symbol |
| **Code Action** | `Ctrl+.` or `<leader>ca` | Quick fix / refactor |
| **Format** | `Shift+Alt+F` | Format document |

---

## Notes

- **`<leader>`** = `Space` (by default in Vim)
- **`Ctrl+`** = Hold Control + key
- **`Alt+`** = Hold Alt + key
- **`Shift+`** = Hold Shift + key

So `<leader>rn` means: Press `Space`, release, then press `r`, then `n`
